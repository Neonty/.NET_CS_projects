
Бұл жобаның мақсаты – пайдаланушыға туылған күніне сәйкес жұлдыздық белгісін анықтап, 2025 жылға арналған жорамалын көрсету. Бағдарлама тек қана негізгі зодиак белгілерін емес, сонымен қатар Қытай жылнамасы бойынша туған жыл жануарын есептеп, бақытты санын анықтауға мүмкіндік береді.

Жобаның негізгі міндеттері:
- Пайдаланушының туған күнін енгізу арқылы жұлдыздық белгі мен Қытай жылнамасы жануарын анықтау;
- 2025 жылға арналған жорамал мәтінін көрсету;
- "Бақытты санын" есептеу;
- Нәтижелерді визуалды түрде көрсету және сақтау мүмкіндігін қамтамасыз ету.

Бағдарлама көңіл көтеру үшін ғана емес, сонымен қатар күнтізбелік есептеулер, алгоритмдік ойлау және графикалық интерфейсті қолдану тәжірибесін береді.

## Жобаның құрылымы және қолданылған әдістер

### Пайдаланушы интерфейсі
Бағдарлама **Windows Forms** технологиясында жасалған. Пайдаланушы интерфейсі қарапайым және ыңғайлы:

- **DateTimePicker** – туған күнді енгізу үшін;
- **Button** – әр функцияны орындау үшін (`button1` – жұлдыздық белгі, `happy` – бақытты сан, `china` – Қытай жылнамасы жануары);
- **Label** – нәтиже мен қосымша ақпаратты көрсету;
- **PictureBox** – жұлдыздық белгінің немесе жануардың суретін көрсету.

Контролдардың визуалды көрінісі дөңгелектенген (`RoundControl()`), нәтижелер экран ортасына орналастырылады.

```csharp
private void RoundControl(Control c, int radius)
{
    var path = new GraphicsPath();
    path.StartFigure();
    path.AddArc(new Rectangle(0, 0, radius, radius), 180, 90);
    path.AddArc(new Rectangle(c.Width - radius, 0, radius, 90));
    path.AddArc(new Rectangle(c.Width - radius, c.Height - radius, radius, 90));
    path.AddArc(new Rectangle(0, c.Height - radius, radius, 90));
    path.CloseFigure();
    c.Region = new Region(path);
}

```

### Жұлдыздық белгі және болжам алгоритмі

Жұлдыздық белгіні анықтау үшін `GetZodiac()` функциясы қолданылады. Ол ай мен күн бойынша дұрыс белгіні қайтарады:

```csharp
private string GetZodiac(DateTime birthDate)
{
    int day = birthDate.Day;
    int month = birthDate.Month;
    switch (month)
    {
        case 1: return (day <= 19) ? "Тауешкі" : "Суқұйғыш";
        case 2: return (day <= 18) ? "Суқұйғыш" : "Балықтар";
        // ... қалған айлар
        default: return "";
    }
}

```

Әр белгіге арналған 2025 жылға болжам мәтіндері `Dictionary<string, string>` құрылымында сақталған, сондықтан мәтінді тез шығару мүмкіндігі бар.

### Бақытты сан және Қытай жылнамасы

**Бақытты сан** туған күннің цифрларының қосындысы арқылы есептеледі, кейін бір цифрлық санға дейін қысқартылады:

```csharp
private static int GetLuckyNumber(DateTime date)
{
    int sum = 0;
    foreach (char c in date.ToString("ddMMyyyy"))
        sum += c - '0';
    while (sum > 9)
    {
        int temp = 0;
        while (sum > 0) { temp += sum % 10; sum /= 10; }
        sum = temp;
    }
    return sum;
}

```

**Қытай жылнамасы жануары** туған жылға сәйкес анықталады:

```csharp
private string GetChineseZodiac(DateTime birthDate)
{
    string[] animals = {"Тышқан", "Бұқа", "Жолбарыс", "Қоян", "Айдаһар", "Жылан", "Жылқы", "Ешкі",
                        "Маймыл", "Әтеш", "Ит", "Доңыз"};
    int index = (birthDate.Year - 1900) % 12;
    if (index < 0) index += 12;
    return animals[index];
}

```

-   Жұлдыздық белгінің суреттері **PictureBox** арқылы көрсетіледі;

-   Нәтижелер экран ортасына орналастырылады, қолданушы интерфейсі ыңғайлы.

### Нәтижені сақтау

Пайдаланушы нәтижені мәтін файлына сақтай алады:

```csharp
private void SaveAsTxt(string text)
{
    SaveFileDialog sfd = new SaveFileDialog();
    sfd.Filter = "Text file (*.txt)|*.txt";
    sfd.FileName = "prediction.txt";
    if (sfd.ShowDialog() == DialogResult.OK)
    {
        File.WriteAllText(sfd.FileName, text, Encoding.UTF8);
        MessageBox.Show("Файл сәтті сақталды!");
    }
}

```

## Қорытынды

Жоба пайдаланушыға жеке болжамды жылдам, ыңғайлы және визуалды тартымды түрде ұсынуға мүмкіндік береді. Бағдарлама келесі тәжірибелерді береді:

-   Күнтізбелік есептеулер мен алгоритмдік ойлау;
    
-   Деректер құрылымдарын қолдану (`Dictionary`);
    
-   Графикалық интерфейсті жасау және элементтерді өңдеу;
    
-   Файлмен жұмыс жасау және нәтижені сақтау.
    

Бұл бағдарлама тек көңіл көтеру үшін ғана емес, сонымен қатар бағдарламалау негіздерін, GUI компоненттері мен алгоритмдік есептеулерді практикада қолдануға мүмкіндік береді.
